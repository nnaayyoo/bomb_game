# **게임 시작 전**

## Game set
-플레이어의 이름을 쉼표로 구분하여 최대 4명까지 입력 합니다
-쉼표로 구분하지 않거나 1명 이하, 4명 초과의 인원을 입력하면 제입력창이 뜹니다

## 게임시작 후

### 게임 방법
-먼저 선을 정하고, 선부터 시계방향으로 돌아가면서 턴을 진행합니다.
    자기 턴에는 카드덱에서 카드를 뽑거나, 자신이 들고 있는 카드를 사용할 수 있습니다.
    카드덱에서 카드를 뽑으면 1턴이 소모되고, 카드를 사용하는 것은 턴을 소모하지 않습니다.(특정 카드 제외)   
### 턴 시작
 - 각 턴이 시작되면  0 또는 1, show, help를 선택하여 행동합니다
    - 0 : 카드를 뽑은 후 턴을 교체합니다
    - 1 : 보유중인 카드를 사용합니다
    - show : 보유중인 카드를 확인합니다
    - help : 도움말을 확인합니다
   
### 앞선 턴 시작에서 1선택시 턴을 진행합니다
  - 각 카드를 사용해 게임을 진행합니다 [참고](https://github.com/Dobbyyyyyyyy/Bomb-Game/blob/main/README.md#%EC%B9%B4%EB%93%9C-%EC%83%81%EC%84%B8%EC%84%A4%EB%AA%85)
